# BY Project

This is the root README for the BY project.

## Project Structure

- cli/ - Command line interface tools
- desktop/ - Desktop application
- eslint-i18n-wrap/ - ESLint plugin for i18n wrapping

## Getting Started

### Prerequisites
- Node.js 18+
- pnpm

### Installation
```bash
pnpm install
```

### Usage
Run the CLI:
```bash
pnpm -F cli start
```

Start the desktop app:
```bash
pnpm -F desktop dev
```
